\section{论文概览}

\subsection{核心问题}

WildDet3D 研究开放世界单目 3D 目标检测：给定一张普通 RGB 图像，以及文本、点击、2D 框或示例框等提示，模型需要预测目标的三维位置、物理尺寸、空间朝向和置信度。它要回答的不只是“物体在图像中的哪里”，还包括“物体在真实三维空间中的哪里、大小如何、朝向如何”。

单目 3D 检测的根本困难是深度缺失和尺度歧义。一个近处小物体与远处大物体可能在图像中投影相似，仅靠 RGB 外观很难恢复真实尺度。WildDet3D 的切入点是：以 RGB 为主输入，允许在有条件时接入深度、LiDAR、ToF 或双目等额外几何线索；没有外部几何时，也能回退到单目估计路径。

\subsection{一句话总结}

WildDet3D 是一个面向开放世界的 promptable 3D detection 框架：它统一文本、点、框和示例提示，把 2D 查询提升为 3D 框，并通过可插拔的深度/几何后端缓解单目 3D 检测中的尺度与空间定位不确定性。

\subsection{现有方法的局限}

\begin{itemize}
    \item 提示形式单一。已有方法往往只支持文本查询、点提示或 oracle 2D 框中的一种，难以适配真实交互系统中“用户可能以不同方式指定目标”的需求。
    \item 几何信息利用不足。很多单目 3D 检测方法默认只使用 RGB，即使外部深度或局部 LiDAR 可用，也缺少统一、稳定的融合接口。
    \item 数据分布狭窄。传统 3D 数据集多来自自动驾驶或室内场景，类别集中在车、人、家具等有限对象，难以支撑开放世界长尾类别的 3D 泛化。
\end{itemize}

\subsection{主要贡献}

\begin{enumerate}
    \item 提出统一的开放世界 3D 目标理解框架，支持从任意单张图像中预测目标 3D 框。
    \item 设计多提示输入机制，统一处理文本、点、2D 框和示例框等不同交互形式。
    \item 引入可插拔几何后端，在有深度时增强几何估计，在无深度时退化为单目模式。
    \item 构建大规模开放世界 3D 数据，覆盖约百万级样本和大量长尾类别，为 in-the-wild 3D 检测提供数据支撑。
\end{enumerate}
